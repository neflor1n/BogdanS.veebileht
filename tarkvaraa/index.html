Mis tahendab MVP?

Minimaalne elujõuline toode on toote, teenuse või teenuse testversioon, millel on minimaalne funktsioonide kogum (mõnikord isegi üks), mis pakub lõpptarbijale väärtust.

Kuidas luua Git remote repo?

Järgige neid samme, et luua Git remote repository ja ühendada oma kohalik repository:

1. Samm: Loo kaugrepo

Minge GitHubi (või mõnda muud teenust, nagu GitLab või Bitbucket), et luua uus repository. Täitke järgmised sammud:

  • Logige sisse oma Git teenusesse (näiteks GitHubi).
  • Klõpsake nuppu "New" uue repository loomiseks.
  • Täidake repository nimi, kirjeldus ja valige, kas see on avalik või privaatne.
  • Klõpsake "Create repository" (ärge lisage README faili veel).
